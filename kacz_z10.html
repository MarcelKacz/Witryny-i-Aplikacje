SEKTORY

Selektory są elementami arkusza CSS.

Postać ogólna stylu:

selektor { cecha: wartość; cecha: wartość; ...; }

Każda reguła odnosi się do konkretnego elementu (znacznika) i składa się z dwóch części:

  • selektora
  • deklaracji

Selektor określa, do jakich elementów ma zostać przypisane formatowanie, np. p, h1.

Deklaracja jest umieszczona w nawiasie klamrowym {...} i podaje formatowanie elementu.

Każda deklaracja składa się z:

  • przynajmniej jednego zespołu cecha (inaczej własność lub właściwość – ang. property)
  • wartości (ang. value), przy czym można podać dowolną liczbę zespołów cecha–wartość, rozdzielając je średnikiem (;)

Średnik (;) na końcu deklaracji nie jest konieczny.

Każda grupa elementów (znaczników np. <body>, <p>, <b>) ma określony zespół cech CSS, które można jej przypisać.

Każda cecha ma ściśle wyszczególnioną listę wartości, które może przyjąć.

Na przykład cecha text-align może być przypisana tylko do elementów blokowych i może przyjmować wartości: left, right, center, justify.

Przykład 1:

p { color: red; }

Selektorem jest znacznik p,
deklaracją jest { color: red },
cechą jest color,
wartością jest red.
Przykład 2:

p b { color: red; background-color: yellow }

Selektorem jest zestawienie znaczników p b,
deklaracja zawiera dwa zespoły cecha–wartość.

Uwaga: W przeglądarkach MSIE 8.0 i starszych jeden plik arkusza CSS może zawierać tylko 4095 selektorów. W przypadku przekroczenia tej liczby konieczny jest podział arkusza CSS na kilka osobnych plików.
